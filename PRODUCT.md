# Product context

## Register

**brand** (marketing portfolio surface): the design is the product. Showcase of shipped apps with a cohesive black-and-gold, editorial identity aimed at Japanese learners and a premium first impression.

## Users

- Prospective users discovering JLPT and Japanese study tools on mobile and desktop.
- Developers or partners scanning a minimal catalog of apps the author builds.

## Product purpose

Single-origin portfolio on GitHub Pages: list all apps in one calm grid, then a detail page per app (starting with the JLPT app) that mirrors a refined, shiri-inspired hero, features, and store CTAs without clutter.

## Brand and tone

- Professional, restrained luxury; cultural depth aligned with Japanese learning (serif + clean sans).
- No hype copy; short labels and one strong line per screen.
- Anti-reference: neon crypto vibes, generic purple-gradient SaaS heroes, dense dashboards as marketing.

## Strategic principles

- Black–gold palette with metallic gold used sparingly for hierarchy and borders.
- Generous whitespace, readable line length, motion that supports scroll (fade/slide), never layout-thrash.
- Static-first Vite build; router + `base` documented for GitHub Pages project sites.
